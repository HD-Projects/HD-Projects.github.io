Why we use GitHub

We use GitHub to host all of our open source projects so that anyone in the world can make suggestions see our code and copy it. We also use GitHub to track releases and changes to our code for every change.

We have found that useing GitHub has been very benifical to us as we have not lost any code since. Whit GitHub we can also have access to all of our code from any browser.
